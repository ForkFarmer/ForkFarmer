# ForkFarmer
Java GUI Frontend for XCH Forks.

Discord: https://discord.gg/Mx9ZNHta

Requires JRE 1.8 or greater to run: https://www.java.com/en/download/

Uses JTatto look-and-feel: http://jtattoo.net/
